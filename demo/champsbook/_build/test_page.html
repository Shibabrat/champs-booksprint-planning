Text formatting

  • Normal text: hello world only produces: hello world
  • Italic: _hello world_ or *hello world* will result in hello world
  • Bold: __hello world__ or **hello world** will result in hello world
  • Strike-through: ~~hello world~~ will result in hello world
  • Code-like looking: `f = my_function(a,b)` will result in f = my_function(a,b)
  • Coloured text: <span style="color:red">hello world</span> will result in hello world
  • Blockquotes

Blockquotes are very handy in email to emulate reply text. This line is part of the same quote.

| Tables        | Are             |  Cool  |

Adding Equations

How to add equations

There are fundamentally two ways to add equations.

Method A Turn a cell into a Markdown cell and simply copy and paste your code as in LaTex. For example, let's write Maxwell's equations

\begin{aligned}
\nabla \times \vec{\mathbf{B}} -\, \frac1c\, \frac{\partial\vec{\mathbf{E}}}{\partial t} & = \frac{4\pi}{c}\vec{\mathbf{j}} \\
\nabla \cdot \vec{\mathbf{E}} & = 4 \pi \rho \\
\nabla \times \vec{\mathbf{E}}\, +\, \frac1c\, \frac{\partial\vec{\mathbf{B}}}{\partial t} & = \vec{\mathbf{0}} \\
\nabla \cdot \vec{\mathbf{B}} & = 0
\end{aligned}

If we copy and paste the above into a Markdown cell, and press Crtl+ENTER, then we get

\begin{align} \nabla \times \vec{\mathbf{B}} -\, \frac1c\, \frac{\partial\vec{\mathbf{E}}}{\partial t} & = \frac{4\pi}{c}\vec{\mathbf{j}} \\ \nabla \cdot \vec{\mathbf{E}} & = 4 \pi \rho \\ \nabla \times \vec{\mathbf{E}}\, +\, \frac1c\, \frac{\partial\vec{\mathbf{B}}}{\partial t} & = \vec{\mathbf{0}} \\ \nabla \cdot \vec{\mathbf{B}} & = 0 \end{align}

Inline equations can be added to Markdown cells like in LaTex.

For example, say we want to inser this inline equation $c = \sqrt{a^2 + b^2}$

Itemising

Suppose we want to write a lists of elements. In LaTeX, we would usually do:

\begin{itemize}
    \item My First item
    \item My Second item
    \item My Third item
\end{itemize}

Unfortunately, if we tried the two methods shown above, this will fail. Instead, we will get somehting ugly as shown below.

\begin{itemize} \item My First item \item My Second item \item My Third item \end{itemize} 
%%latex
\begin{itemize}
    \item My First item
    \item My Second item
    \item My Third item
\end{itemize}
\begin{itemize} \item My First item \item My Second item \item My Third item \end{itemize}

Markdown method In Markdown, we need to do instead

* My First item
* My Second item
* My Third item

If you copy and paste the lines above in a Markdown cell, then you get

  • My First item
  • My Second item
  • My Third item

Similarly, we can do something similar when enumerating items. Example:

  1. My First item

  2. My Second item

  3. My Third item

Further indexing

* My First item
    * hi
        * hi again
            * hi again and again
* My Second item
* My Third item

this produces

  • My First item
    • hi
      • hi again
        • hi again and again
  • My Second item
  • My Third item

Equation numbering/referencing

NOTE 1: This is currently possible using the MathJax Equation Numbering. This can be installed according to instructions provided here.

A working example is shown next.

First Say we want to number this equation

\begin{equation}
\nabla \times \vec{\mathbf{B}} -\, \frac1c\, \frac{\partial\vec{\mathbf{E}}}{\partial t} = \frac{4\pi}{c}\vec{\mathbf{j}} 
\end{equation}

That produces this output

\begin{equation} \nabla \times \vec{\mathbf{B}} -\, \frac1c\, \frac{\partial\vec{\mathbf{E}}}{\partial t} = \frac{4\pi}{c}\vec{\mathbf{j}} \label{eq:Maxwell_1} \end{equation}

In Markdown is not different. We can label equations like in LaTeX. So, we just add a label, like \label{eq:Maxwell_1}

\begin{equation}
\nabla \times \vec{\mathbf{B}} -\, \frac1c\, \frac{\partial\vec{\mathbf{E}}}{\partial t} = \frac{4\pi}{c}\vec{\mathbf{j}}
\label{eq:Maxwell_1}
\end{equation}

Second Now, suppose that you want to reference this equation in your text

In LaTeX, to reference the equation normally we do:

Hey! Look at this pretty equation \ref{eq:Maxwell_1}

BUT, in Markdown we actually do:

Hey! Look at this pretty equation $\eqref{eq:Maxwell_1}$

which procudes a hyperlink to the referenced equation.

Hey! Look at this pretty equation \eqref{eq:Maxwell_1}

Method X Using HTML attributes

FIRST Convert .bib file into a formated .html file using nbconvert.

This approach allows us to insert citations in Markdown cells using HTML data attributes.

Installation instructions for nbconverter can be found here. This can be easily installed with conda though, a number of dependencies need to be considered. See instructions.

Example: We want to turn a file like this into something like this

For instance, a bib entry
